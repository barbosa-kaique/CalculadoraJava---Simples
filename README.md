# Tecnologias utilizadas:

- Java
- JavaFX

<br>

# Como funciona

- Os números digitados aparecem no TextField (visor)
- Ao clicar em uma operação, o primeiro número é armazenado
- O resultado é calculado ao clicar no botão "="
- O histórico da conta é exibido acima do visor
